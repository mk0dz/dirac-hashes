Dirac Hashes Documentation

Introduction

Dirac Hashes is a quantum-resistant cryptographic library designed to provide secure hashing and signature algorithms that can withstand attacks from quantum computers.

Installation

pip install dirac-hashes

Hash Functions

DiracHash

A quantum-resistant hash function based on quantum mechanical principles.

from quantum_hash.dirac import DiracHash

# Create a hash object
hash_obj = DiracHash()

# Hash a message
message = b"Hello, quantum world!"
hash_value = hash_obj.hash(message)

# Verify the hash
print(f"Hash: {hash_value.hex()}")

Digital Signatures

DiracSignature

A quantum-resistant digital signature scheme.

from quantum_hash.signatures import DiracSignature

# Generate a key pair
signature = DiracSignature()
public_key, private_key = signature.generate_keypair()

# Sign a message
message = b"Sign this quantum-resistant message"
signature_value = signature.sign(message, private_key)

# Verify the signature
is_valid = signature.verify(message, signature_value, public_key)
print(f"Signature valid: {is_valid}")
